# 📊 Maven Market Power BI Project

## 📌 Project Overview
This project analyzes the Maven Market retail dataset using Microsoft Power BI. 
The goal is to evaluate sales performance, profit trends, customer behavior, and overall business insights.

## 🎯 Objectives
- Analyze Total Revenue, Total Cost, and Total Profit
- Calculate Profit Margin using DAX
- Track monthly and yearly sales trends
- Identify top-performing products and customers
- Create interactive dashboards with filters and bookmarks

## 🛠 Tools & Technologies Used
- Microsoft Power BI
- DAX (Data Analysis Expressions)
- Data Modeling
- Data Visualization
- GitHub

## 📊 Key Metrics
- Total Transactions
- Total Revenue
- Total Cost
- Total Profit
- Profit Margin %

## 📈 Features
- KPI Cards
- Matrix Visual
- Interactive Charts
- Slicers & Filters
- Bookmark Navigation
- Calculated Columns & Measures

## 🚀 Conclusion
This project demonstrates strong data analysis, dashboard design, and business intelligence skills using Power BI.
